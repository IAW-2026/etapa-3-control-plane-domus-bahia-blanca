# control-plane

Aplicación **Control Plane** del [Proyecto IAW 2026](https://iaw-2026.github.io/proyecto/) — comisión `Domus Bahia Blanca`.

Panel administrativo centralizado (Etapa 3): gestión unificada de todas las apps, activación/desactivación de usuarios y resolución de disputas.

---

Deploy: <https://etapa-3-control-plane-domus-bahia-bla.vercel.app>

## Paneles del sistema

- **Vendedores** — Lista de vendedores inmobiliarios. Muestra nombre, email, agencia y teléfono. Permite activar, desactivar o eliminar vendedores.

- **Propiedades** — Lista de propiedades organizadas por estado (publicadas, borradores, archivadas). Muestra título, precio, dirección, tipo y operación. Permite publicar borradores, archivar publicadas, republicar archivadas y eliminar. Incluye filtros por tipo, operación, zona, agencia, moneda y rango de precio.

- **Clientes** — Lista de compradores registrados. Muestra sus datos y permite gestionar roles y cuentas.

- **Agentes** — Gestión de agentes inmobiliarios. Muestra nombre, email, teléfono e inmobiliaria asociada. Permite editar sus datos y cambiar su estado (pendiente, aceptado, rechazado).

- **Pagos** — Registro de suscripciones de vendedores. Muestra plan, límite de publicaciones, fechas y estado. Permite cambiar el estado entre activa, pendiente de pago, inactiva o vencida.

- **Visitas** — Turnos de visita a propiedades. Muestra comprador, agente, propiedad, fecha y estado. Permite confirmar, cancelar o eliminar visitas.

- **Reseñas** — Moderación de opiniones de clientes. Muestra usuario, propiedad, puntuación, comentario y respuesta del vendedor. Permite eliminar reseñas o respuestas.
